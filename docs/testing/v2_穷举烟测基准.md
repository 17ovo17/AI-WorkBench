# AI WorkBench v2 穷举烟测基准

> 版本：v2.0
> 编制日期：2026-04-29
> 适用范围：AI WorkBench 端到端穷举烟测（API + 工作流 + 知识库 + 前端 + 告警闭环）
> 用例编号格式：`QA-V2-{模块}-{序号}`
> 用例总量：195 条（10 章节）

## 文档说明

本文档是 AI WorkBench v2 端到端测试的"穷举烟测基准"。每个章节针对一个核心模块，列出可执行、可验证、可归档证据的具体测试用例。所有用例的执行结果汇总至 [`v2_测试报告.md`](./v2_测试报告.md)。

每条用例的标准格式：

```
QA-V2-{模块}-{序号} | {用例标题}
步骤：{接口/页面/操作}
预期：{断言}
实际：{执行结果}
证据：.test-evidence/v2-{ts}/{文件名}
状态：PASS | FAIL | BLOCKED | NOT_RUN | RISK
```
